CONTROLLO RESISTENZA ELETTRICA PIANTE BIO-SENSING · OHM · DATA → VISUAL/AUDIO
// PLANT BIO-RESISTANCE · INTERFACCIA
Le piante come potenziometro
Installazione di lettura continua della resistenza elettrica del tessuto vegetale (R = V/I, misurata via partitore o INA125/ADS1115 su MCU). La variazione di Ω nel tempo — dipendente da umidità, conducibilità della linfa, stress cellulare — viene mappata su parametri di output (luce, suono, posizione di un attuatore) creando un canale di comunicazione diretto pianta → macchina.

// FLUID SPLITTER · BIFURCATION
Un emettitore, molti rami

CONTROLLO VISUAL & AUDIO DA TEKKEN 3 CONTROLLER PS1 · HACK · MIDI / OSC · VJ + AV
// TEKKEN 3 CONTROLLER · HARDWARE HACK
Il combo come partitura
Re-routing del controller PS1 originale di Tekken 3 verso un'interfaccia MIDI / OSC. Le sequenze di tasti, le combo, gli stick analogici diventano vettori di controllo per parametri visuali (mappatura su shader, filtri, generatori) e parametri audio (synth, sample, effetti). L'estetica gestuale del fighting game si trasforma in una performance audiovisuale.
